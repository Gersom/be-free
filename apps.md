# Aplicaciones

## Utiles

- Redshift: programa para aliviar el cansancio ocular.

  * instalacion
  ```sh
  $ sudo add-apt-repository ppa:jonls/redshift-ppa
  $ sudo apt-get update && sudo apt-get install redshift
  ```

  * uso  
  ```sh
  $ redshift -l -18.010529:-70.240198
  ```
